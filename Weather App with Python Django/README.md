# 🌦️ Weather App – Django Weather Forecast System

A modern Django-based real-time weather forecasting web application with full Docker support.  
This project demonstrates API integration, clean UI design, and containerized deployment using Docker.

---

## 🚀 Features

- 🌍 Real-time weather forecasting for any city
- 🔎 City-based search functionality
- 🌡️ Temperature, humidity, wind speed, and weather conditions
- 🎨 Clean and responsive user interface
- 🔗 API-based weather data integration
- 🐳 Fully containerized using Docker & Docker Compose

---

## 🧰 Tech Stack

- Python 3
- Django
- SQLite3
- HTML5, CSS3, Bootstrap
- Docker
- Docker Compose
